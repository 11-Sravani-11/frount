!DOCTYPE HTML

It's a short instruction telling web browsers "This is an HTML5 document!" to ensure proper rendering.

HTML

HTML is the standard language for creating and structuring web pages and their content for display in web browsers.

Head

The head section in HTML contains meta-information about the webpage, like its title and links to stylesheets, which isn't directly visible on the page itself.

body

The body section in HTML contains all the visible content of a webpage, such as text, images, and videos.

Footer

The footer section in HTML typically contains copyright information, contact details, or links at the bottom of a webpage.

br

br is an HTML element that creates a line break, moving the following text to the next line.

hr

hr is an HTML element that creates a horizontal rule, which is a thematic break or a horizontal line across the content.
